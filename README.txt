nesemu
A simple Nintendo Entertainment System emulator written in C++ for SENG475

Built by:

Quickstart
$ cmake -H. -Bbuild -DCMAKE_INSTALL_PREFIX=$INSTALL_DIR/
$ cmake --build build --clean-first --target install
$ ./$INSTALL_DIR/bin/demo
